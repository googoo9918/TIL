## Port
- 포트번호는 대상 IP 기기의 특정 어플리케이션을 특정하는 번호
  - TCP와 UDP 모두 포트번호 사용
- ![image](https://user-images.githubusercontent.com/102513932/193193632-3eddf741-ab4d-41f6-a070-0b8575477d5a.png)
  - IP 주소만으로는 서버를 특정 지을 수 없음
    - 포트 번호를 사용해 receiver을 특정함
