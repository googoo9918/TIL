### 유효숫자
- 0이 아닌 숫자는 모두 유효숫자임
  - 1.23
    - 유효숫자 3개: 1, 2, 3
- 0이 아닌 숫자 사이의 0은 유효숫자임
  - 3.002
    - 유효숫자 4개: 3, 0, 0, 2
- 소수점 아래 0이 아닌 숫자 뒤의 0은 유효숫자임
  - 2.40
    - 유효숫자 3개: 2, 4, 0
- 자연수에서 끝의 0은 유효숫자인지 알 수 없음
  - 500
    - 0 두 개는 유효숫자인지 알 수 없음
