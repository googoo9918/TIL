## 개인회원 이용금액의 이해 및 - 마케팅
### 이용금액 톺아보기
- 이용 금액
  - 유실적회원수(가용회원수 * 유실적률) * 인당 이용금액
- 카드 회원 구분
  - 유효회원: 유효기한 남아있는 신용카드 보유
  - 가용회원: 사용 가능한(거래정지가 등재되지 않은) 카드 보유 회원
  - 유실적회원: 가용회원 중 당월 이용실적이 있는 회원
  - 핵심활동고객: 유실적회원 중 당월 이용실적이 100만원 이상인 회원
- 리테일은 CA(Cash Advance, 현금서비스)는 제외
  - 개인 신판, 개인 체크, 기업카드(신판 + 체크)
- 일반매출: 리테일 이용금액에서 전략매출(저수익매출: 국세, 지방세, 4대보험) 제외
- 기준일의 이해
  - 재무 기준 = 내부 기준
    - 월말 비영업일이면 다음달로 이월
  - 협회 기준 = 대외 기준, 공시 기준
    - 1일 ~ 말일
- 용어 정리
  - LS(Lump Sum, 일시불), IS(Installment, 할부), CA(Cash Advance, 현금서비스)
    - 앞에 R이 붙으면 Revolving
  - YTD: 연초 누계 / MTD: 월초 누계
  - YoY: 전년 대비 / MoM: 전월 대비 / QoQ: 전분기 대비
  - 모음 또는 ㄴ받침이면 -> 율

### 전략가맹점마케팅 톺아보기
- 전략가맹점마케팅
  - 전사 지표 변동에 영향력이 높은 전략가맹점 선정하여 진행

## 가맹점 제도 및 프로세스의 이해
### 개요
- 카드사, 회원, **가맹점** 으로 이루어진 3당사자 체제
- 가맹점에서 발생하는 여러 카드사와의 거래에 대한 지원업무를 **VAN**사가 다행

### 가맹점 업무 흐름도 
- 가맹점 모집 시 준수
  - 사업장 방문하여 영업 여부 확인
- 신용카드 결제금지대상
  - 자본시장과 금투업에 관한 법률에 따른 금융 투자상품
    - 보험 상품은 예외임

### 가맹점 가입
- 가맹점 모집용역ㅈ제도
  - 가맹접 모집 VAN으로 위임
  - 계약요청 -> 서류징구 -> 계약체결 -> 가맹점모집
- 가맹접 가입 신청/모집
  - 모바일(태블릿) 접수
    - 99.9%가 모집인 채널로 가입 신청
- 가맹점 가입_심사
  - 가맹점 정보입력은 카드사 공동 수행 

### 승인 / 매입
- MS 거래는 거래 제한
  - IC 칩 손상 -> FALL BACK 거래
- 전표매입
  - EDI
    - 대형가맹점, 직접 매입내역을 카드사로 전송
    - EDI 중계 사업자가 있긴함
    - 직라인이 진짜 바로 보내는거
  - DDC
    - VAN사가 전송
    - 단, 실물 매출전표를 수거하여 보관
  - DSC
    - VAN사가 전송
    - 단, 실물전표 대비 전자서명 수거(Signature Capture)

### 가맹점 신용판매대금 지급
- 중소 가맹점은 매입+1 영업일 기준으로 대금지급 해야함
- 가맹점 수수료
  - 공통원가 + 개별원가 + 마진
  - 공통원가
    - 자금조달비용, 위험관리 비용, 일반관리비
  - 개별원가
    - 마케팅비용, VAN 비용, 조정비용 등
- 적격비용 재산정 -> 6년 주기로 변경

### 가맹점 사후관리
- 가맹점 정보변경 주로 모집인을 통함
- 가맹점 해지
  - 장기 무실적(5년) 가맹점 개별 해지 처리
- 공용전표특약
  - 가맹점이 모든 카드사의 매출을 공동으로 사용할 수 있는 공용전표를 직접 조제/사용

## 채권관리의 이해
### 채권관리 개념 이해
- 채권잔액
  - 연체원금 + 미청구원금
  - 보유한 채권의 총 잔액임!!(어차피 나중에도 못갚을 거잖아)
- 축소금액
  - 실제 회수 금액 + 회수에 의해 정상화된 변제기 미도래 금액
- 축소욜
  - 축소금액/연체채권잔액
- 연체원금 100, 미청구원금 900이라 하자
  - 50 상환시 -> 50/1000
    - 축소율 5%, 축소금액==회수금액==50
  - 100 상환시 -> 연체 상태 아님
    - (100 + 900) / 1000
    - 축소금액=1000, 회수금액==100

### 복습
- 회수위임
  - 채권의 회수업무를 외부 신용정보회사(추심회사)로 위임
- 특수채권
  - 대손상각 처리된 채권, 별도 편입절차를 거친 채권임
- 대손상각
  - 회수가 불가능하거나, 회수실익이 없는 것으로 인정되는 채권을 회계상 손실 처리하는 것
  - 통산 연체 6개월이상 채권
